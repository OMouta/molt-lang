# Editor Support

## VS Code

A VS Code extension scaffold for `.molt` files lives in:

[`editors/vscode-molt`](../editors/vscode-molt)

It includes:

- syntax highlighting
- language configuration for comments, brackets, and auto-closing pairs
- a custom `.molt` file icon

### Try It Locally

1. Open `editors/vscode-molt` in VS Code.
2. Press `F5` to launch an Extension Development Host.
3. Open a `.molt` file in the new window.
